Описание программы

Программа разработана в рамках расчетно-графической работы по дисциплине «Теория формальных языков и компиляторов».

Техническое задание:

Разработать приложение – текстовый редактор, дополненный функциями языкового процессора (выполнить программную реализацию алгоритма лексического анализа (сканера)).

Приложение имеет графический интерфейс пользователя.

Язык реализации: C#

Текстовый редактор включает следующие элементы:

  1. Основное меню программы (Файл, Правка, Текст, Пуск, Справка);
  2. Панель инструментов:
    • Файл – создать, открыть, сохранить;
    • Правка – отменить, повторить, вырезать, копировать, вставить;
    • Пуск – запуск сканера текста;
    • Вызов справки – справочная система с описанием всех функций меню.
  3. Окно/область редактирования текста:

    Редактор текста, в котором работают команды меню «Файл» и «Правка».

  4. Окно/область вывода результатов:

    Отображает сообщения и результаты лексического анализа. Ввод в это поле запрещён.

Дополнительные возможности интерфейса:

  • Изменение размеров текста в окне редактирования и окне вывода результатов;
  • Нумерация строк в окне редактирования текста;
  • Открытие файла при перетаскивании иконки в окно программы (Drag'n'Drop);
  • Строка состояния с текущей информацией о работе приложения;
  • Интерфейс с вкладками для работы с разными модулями (в окне вывода);
  • Отображение результатов анализа в виде таблицы;
  • Горячие клавиши для быстрых команд.
