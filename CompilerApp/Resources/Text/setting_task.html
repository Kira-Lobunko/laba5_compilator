1 Постановка задачи

Прототипом функции в языке C/C++ называется строка объявления, содержащая тип возвращаемого значения, имя функции и (опционально) список параметров с их типами, но не содержащая тела функции. Прототип сообщает компилятору о существовании функции и о том, как она должна вызываться.

Формат записи прототипа функции представлен ниже:

“тип_возвращаемого_значения имя_функции(список_параметров);”

Примеры:

1) Прототип с типом string и без параметров: “string printHello();”

2) Прототип с типом int и одним параметром: “int sum(int a);”

3) Прототип с типом float и двумя (или несколькими) параметрами: “float calculateRectangleArea(float length, float width);”

В связи с разработанной автоматной грамматикой G[<PR>] лексический анализатор (сканер) прототипов функций будет считать верными следующие записи:

1) “bool flag();”

2) “char getFirstChar(string text);”

3) “int add(int a, int b);”

4) “float max(float number1, float number2, float number3);”
